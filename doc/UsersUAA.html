Class: UsersUAA

UsersUAA(endPointopt)

UsersUAA — User management and authentication for Cloud Foundry UAA. Supports password, client_credentials, passcode, and authorization_code grant types. Extends CloudControllerBase for shared HttpUtils / HttpStatus / token management.
