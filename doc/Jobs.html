Class: Jobs

Jobs(endPointopt, optionsopt)

Jobs — background job / task management for Cloud Foundry. NOTE: v2 uses '/v2/jobs' while v3 uses '/v3/tasks' (endpoint renamed). Supports both CF API v2 and v3 with automatic version routing.
